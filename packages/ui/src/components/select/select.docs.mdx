# Autocomplete Select

Autocomplete Select fetches its options from defined API endpoint and presents them for the user to select.

```jsx
import { AutocompleteSelect } from '#components/select';
```

<Markdown>{`
| Prop Name           | Type                                | Default            | Description                                                                                                                                          |
| ------------------- | ----------------------------------- | ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| \`getOptionsURL\`   | \`string\`                              |                    | URL from which data will be fetched to provide options.                 |
| \`options\`         | \`string[]\`                            |                    | An array of options that can be used used when \`getOptionsURL\` is not defined. |
| \`onValueChanged\`   | \`(newValue) => void\`             |                    | Callback function triggered when the selected value changes.                                                                                          |
`}</Markdown>

## Autocomplete Default

In default props configuration Autocomplete select fetches its options after a search string changes. After an option is picked the search is disabled and options can no longer be changed.
