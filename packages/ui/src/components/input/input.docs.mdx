#### Clearable Input Props & Methods

<Markdown>{`
| Prop Name           | Type         | Default | Description                                                                 |
| ------------------- | ------------ | ------- | --------------------------------------------------------------------------- |
| onClear             | () => void   |         | Callback function that is called when the clear action is triggered.        |
| isClearable         | boolean      | true    | Determines if the input field is clearable.                                 |
| forceIsClearable    | boolean      |         | Forces the input to be clearable regardless of its content or other states. |
| customClearButton   | ReactElement |         | A custom component to replace the default clear button.                     |                   |                                                                                                                                                                                                                                                                                                                               |
`}</Markdown>

## Autocomplete Input

The `AutocompleteInput` fetches completion suggestions from given URL and displays them in a popover menu navigable by cursor
or keyboard arrows. `offlineSuggestions` is used as a fallback when `getSuggestionsURL` is not defined.

| offlineSuggestions  | string[]                                      |         | An array of suggestions that are used when \`getSuggestionsURL\` is not defined. |
`}</Markdown>
