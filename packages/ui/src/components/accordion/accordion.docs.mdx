### Default expanded value

During initial rendering, it is possible to specify which of the items will be displayed expanded. This is determined by the `defaultValue` tag placed on the Accordion component.

For `type="single"` is defaultValue defined as a string `defaultValue="value-1"`.

In case of `type="multiple"`id dafultValue defined as a string[] `defaultValue={["item-1", "item-2"]}`
